--- layout: post title: Making Python VirtualEnv Even Easier categories: - development - python ---

If you develop multiple Python projects at once then VirtualEnv is an essential tool to make managing packages and keeping conflicts at bay. However, typing
source /some/long/path/name/sucks/bin/activate
to start your virtual environment gets annoying.

Thus, I created Python Quick Env. Once installed and setup, typing pyenv project1 will start up your VirtualEnv and bring you to the project directory.

Now go enjoy an extra 5 seconds of life everytime you start up a VirtualEnv!

Update 02/12/2012
When in a Virtual Env executing pyenv take you back to your project home. Stay tuned for more updates.
