It's been a while but Version 0.2 is finally here. Version 0.2 adds a few usability enchancements that should help immensely. Here is what is included in Version 0.2.

  • Close confirmation so you don't lose work if you accidentally leave the page.
  • Full path of current file shown so you don't get mixed up with files that are named the same.
  • Right click menu for the file browser with:
    • Create new file or directory. Files can be created empty, by upload, or download via a URL.
    • View file outside the editor. Great for viewing and downloading binary files.
    • Rename file or directory.
    • Delete file or directory.
    • Refresh directory listing.

Download the latest release at Git Hub.

For installation and upgrade directions see the wiki.
