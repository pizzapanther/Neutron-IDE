Today, version 11.12 of Neutron IDE is released. This month's release only adds one feature, but it is a biggy. Here is what is included in Version 11.12.

  • Search Tool: The search tool allows you to search your current tab, all open tabs, or a directory. It also includes a replace mechanism.

    The directory search uses Celery to execute tasks asynchronously while the Neutron IDE interface updates your the results as they come in.

    The directory search in particular is a very powerful tool so use it with caution. As said in Spiderman: "With great power comes great responsibility".

Download the latest release at Git Hub (recommended) or download the tarball.

For installation and upgrade directions see the wiki.
